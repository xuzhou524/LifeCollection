##How to build
1)  Clone the repository
```
$ git clone https://github.com/xuzhou524/LifeCollection.git
```
2) Open the workspace in Xcode
```
$ open "LifeCollection.xcworkspace"
```
3) Compile and run the app in your simulator or iOS device

##Requirements
* Xcode 9.0+
* iOS 9+
